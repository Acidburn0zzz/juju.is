What is Juju?

Built to manage complex software architectures wherever they’re run, Juju simplifies deployment and post-install operations like upgrades, updates and configuration management.

Use Juju to manage applications and services on public clouds such as Amazon AWS, Google GCE and Microsoft Azure, on bare metal, or any hybrid cloud configuration.
